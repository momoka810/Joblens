# サーバー起動方法

## 正しい起動方法

### 1. プロジェクトディレクトリに移動

```bash
cd "/Users/momokaiwasaki/Documents/AIエンジニア講座/7-3-2_Dify×Bolt.new/project"
```

### 2. サーバーを起動

```bash
python3 server.py
```

**実行結果の例：**
```
サーバーを起動しました: http://localhost:8000
終了するには Ctrl+C を押してください
```

### 3. ブラウザでアクセス

サーバーが起動したら、ブラウザで以下のURLにアクセス：
```
http://localhost:8000
```

## よくある間違い

❌ **間違い：**
```bash
http://localhost:8000  # これはコマンドではなくURLです
```

✅ **正しい：**
```bash
python3 server.py  # サーバーを起動するコマンド
```

## サーバーを停止する方法

サーバーを停止するには、ターミナルで：
```
Ctrl + C
```

## トラブルシューティング

### ポート8000が使用中の場合

`server.py`の`PORT = 8000`を別のポート番号（例：`8080`）に変更するか、別のターミナルで：

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` にアクセス

### Pythonが見つからない場合

```bash
python server.py
```

または

```bash
python3 server.py
```

## 確認方法

サーバーが正しく起動している場合：
- ターミナルに「サーバーを起動しました」と表示される
- ブラウザで `http://localhost:8000` にアクセスできる
- ページが表示される
